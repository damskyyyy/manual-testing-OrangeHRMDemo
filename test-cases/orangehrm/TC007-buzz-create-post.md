# TC007: Create a new update in Buzz

**Module:** Buzz (Internal Social)
**Priority:** Low
**Test Type:** Functional / Interactive

## Preconditions
- User is logged in.

## Test Steps
1. Click on "Buzz" from the left-hand navigation menu.
2. In the "Update Status" text box at the top of the feed, type: "Hello team! Excited to announce the new web portal is live."
3. Click the "Post" button.

## Expected Result
- The new post should instantly appear at the top of the Buzz newsfeed.
- The timestamp should say "Just now".
- Other users should theoretically be able to see it and "Like" or "Comment" on it.

## Actual Result
- Logged in successfully and navigated to Buzz.
- Posted: "Hello team! Excited to announce the new web portal is live from Antigravity."
- Post appeared successfully at the top of the feed with user "manda akhil user".
